## v5.0.25 - 安装链路修复与稳定性增强

### Bug 修复
- **fix**: 修复 `scripts/install.sh` 在 macOS 上解析 GitHub `tag_name` 失败导致下载 URL 拼接错误的问题
- **fix**: 修复面板一键安装 OpenClaw 时，服务环境缺少 `HOME` 导致 `brew` 报错并安装失败的问题

### 兼容性改进
- **improve**: 任务执行器统一补全关键环境变量（`HOME`/`USERPROFILE`/`PATH`）并去重，提升 Linux/macOS/Windows 后台安装任务稳定性
- **improve**: OpenClaw/Node.js 一键安装在 macOS 下改为官方 Node 二进制安装路径，避免服务场景依赖 Homebrew
