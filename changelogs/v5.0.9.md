# ClawPanel v5.0.9 更新日志

## 🆕 新功能：独立更新工具 (UpdateClawPanel)

### 核心特性
- **独立进程更新**：更新工具运行在独立端口(19528)，与主面板完全隔离，面板停止后更新工具仍可运行
- **临时授权令牌**：仅允许从面板「版本管理」页面跳转访问，5分钟有效期加密令牌，禁止直接URL访问
- **自动化更新流程**：验证授权 → 检测版本 → 停止服务 → 下载更新 → 备份文件 → 替换文件 → 启动服务
- **双线路下载**：优先北京加速服务器，失败自动切换 GitHub，支持断点续传
- **SHA256 校验**：下载完成后自动校验文件完整性，避免替换损坏文件
- **自动回滚**：更新失败（文件替换失败、服务启动失败）自动恢复备份文件并重启旧版本
- **离线更新**：支持手动上传可执行文件进行更新，满足无网络环境需求
- **实时进度展示**：7步进度条 + 实时日志输出，清晰展示更新各阶段状态
- **更新历史记录**：记录每次更新的时间、版本、线路、结果，便于问题排查

### 版本管理页面
- 重新设计「ClawPanel 版本更新」卡片，三栏展示当前版本/最新版本/更新方式
- 「检测新版本」按钮拉取最新版本信息并展示更新日志
- 「前往更新」按钮生成加密令牌并跳转至独立更新页面
- 「更新记录」按钮查看历史更新日志

## 🔧 改进
- **编译产物带版本号**：所有平台编译文件名现在包含版本号（如 `clawpanel-v5.0.9-linux-amd64`）
- **安装脚本适配**：install.sh 和 install.ps1 已适配新的版本号文件名格式
