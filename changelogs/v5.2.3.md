## v5.2.3 - 飞书治理收口、通道配置简化与安装稳定性提升

### 飞书与 Agent 治理
- 合并 PR41，引入飞书单通道 + 插件版本切换、多账号配置、DM 诊断与 `session.dmScope` 指引
- OpenClaw / Agent 保存新增关键数值字段校验，路由预览会按默认账号补全 `accountId`
- WebSocket Hub 增加 graceful shutdown 与慢客户端清理，补充飞书 / 治理 / WebSocket 相关测试

### 通道管理体验
- 简化 `QQ 官方机器人`、`钉钉`、`企业微信（智能机器人）`、`企业微信（自建应用）` 配置表单，只保留高频必填项
- 飞书继续保持单一通道入口，避免官方版 / 社区版双入口带来的配置混乱
- 插件安装在缓存仓库未命中时会自动刷新 registry 再重试，降低 `wecom-app` / `feishu-openclaw-plugin` 等“插件不在仓库中”的误报

### 安装与平台兼容
- Windows 安装 OpenClaw 时增强 Node.js / PATH / 常见目录探测，并在必要时回退官方 MSI 安装
- Skills 页面支持扫描 workspace、agent workspace、OpenClaw work/app 多处技能目录
- Linux 安装脚本支持通过 `CLAWPANEL_SERVICE_USER` 显式指定非 root 服务用户，默认仍保持 root 兼容模式
